This program allows you to enter English vocabulary words in a spreadsheet, and then have APIs (currently only Webster Dictionary) provide definition, pronunciation, etc in the same spread sheet. 

You will need to register with Webster's basic API (free) and receive an API ID.


Step 1: Open a terminal window, and navigate to the Final Directory.
Step 2: Be sure to have a key entered in the first line of WebsterID.txt
Step 3: Be sure to have the xlsx document you wish to process in Final
Step 4: Run $ python3 VocabBuilderUI.py
Step 5: you will be prompted for the file name, include it along with the '.xlsx"
Step 6: You will be prompted for the List name, provide it.
